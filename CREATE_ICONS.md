# Create PWA Icons

Για να δημιουργήσεις τα icons, μπορείς να:

1. Χρησιμοποιήσεις online tool: https://www.pwabuilder.com/imageGenerator
2. Ή να δημιουργήσεις 192x192 και 512x512 PNG files με το logo σου
3. Αποθήκευσε τα ως `public/icon-192.png` και `public/icon-512.png`

Για τώρα, το manifest.json δεν χρειάζεται icons (empty array).
