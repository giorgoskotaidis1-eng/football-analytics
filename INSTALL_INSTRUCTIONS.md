# Οδηγίες Εγκατάστασης

## Επιλογή 1: Double-click στο αρχείο (Πιο Εύκολο!)

1. Βρες το αρχείο `install-jose.bat` στον φάκελο του project
2. Κάνε **double-click** πάνω του
3. Περίμενε να τελειώσει
4. Έτοιμο! ✅

## Επιλογή 2: Terminal/Command Prompt

### Αν είσαι στο VS Code / Cursor:
1. Πάτα `Ctrl + ~` (ή `Ctrl + backtick`) για να ανοίξεις το terminal
2. Γράψε: `npm install`
3. Πάτα Enter
4. Περίμενε να τελειώσει

### Αν είσαι στο Command Prompt (cmd):
1. Άνοιξε το Command Prompt
2. Πήγαινε στον φάκελο του project:
   ```
   cd C:\Users\troll\CascadeProjects\football-analytics-app
   ```
3. Γράψε: `npm install`
4. Πάτα Enter

### Αν είσαι στο PowerShell:
1. Κάνε right-click στον φάκελο `football-analytics-app`
2. Επέλεξε "Open in Terminal" ή "Open PowerShell window here"
3. Γράψε: `npm install`
4. Πάτα Enter

## Τι κάνει το `npm install`?

Εγκαθιστά όλα τα packages που λείπουν, συμπεριλαμβανομένου του `jose` που χρειάζεται για το authentication.

## Μετά την εγκατάσταση:

Μόλις τελειώσει, μπορείς να τρέξεις το project με:
```
npm run dev
```
